⏺ 📊 学术个人网站定制指南                                                                                               
                                                                                                                        
  🎯 快速上手：5步定制流程                                                                                              
                                                                                                                        
  第1步：Fork 并重命名仓库 ⚡                                                                                           
                                                                                                                        
  1. Fork 该仓库到你的 GitHub 账户                                                                                      
  2. 重命名为：YOUR_USERNAME.github.io
  3. 本地克隆：git clone https://github.com/YOUR_USERNAME/YOUR_USERNAME.github.io

  第2步：编辑核心配置 🔧

  编辑 _config.yml 文件，填入你的信息：
  title: "你的名字"
  description: "你的职位 / 研究方向"
  repository: "YOUR_USERNAME/YOUR_USERNAME.github.io"

  author:
    name: "你的全名"
    bio: "简短的个人简介"
    location: "城市，国家"
    email: "your.email@university.edu"
    googlescholar: "https://scholar.google.com/citations?user=YOUR_ID"
    orcid: "0000-0000-0000-0000"
    github: "YOUR_USERNAME"
    twitter: "YOUR_TWITTER"  # 可选

  第3步：更新主要内容 📝

  编辑 _pages/about.md 文件来定制：
  - 关于我：个人简介和研究兴趣
  - 新闻动态：最新公告（职位变动、基金、获奖）
  - 发表论文：使用提供的 HTML 结构添加你的论文
  - 荣誉奖项：你的学术成就
  - 教育背景：你的学历信息
  - 邀请报告：会议演讲
  - 实习经历：企业/学术访问经历

  第4步：替换图片 🖼️

  images/
  ├── avatar.png          # 你的头像照片
  ├── favicon/            # 网站图标文件
  └── [论文图片]          # 你的论文缩略图

  第5步：部署上线 🚀

  git add .
  git commit -m "定制学术个人主页"
  git push origin main
  # 访问：https://YOUR_USERNAME.github.io

  🔧 高级定制

  Google Scholar 自动引用数：

  1. 从以下网址获取你的 Scholar ID：https://scholar.google.com/citations?user=YOUR_ID
  2. 在 GitHub 仓库设置中添加密钥：GOOGLE_SCHOLAR_ID
  3. 在仓库设置中启用 GitHub Actions
  4. 引用数每天自动更新

  颜色方案：

  编辑 _sass/_variables.scss：
  $primary-color: #007bff;      // 主色调
  $secondary-color: #6c757d;    // 次要文本颜色
  $background-color: #ffffff;   // 页面背景
  $text-color: #212529;         // 主文本颜色

  导航菜单：

  编辑 _data/navigation.yml：
  main:
    - title: "关于我"
      url: "/#about-me"
    - title: "发表论文"
      url: "/#-publications"
    - title: "教学"          # 添加/删除版块
      url: "/#-teaching"

  # 或
  bundle exec jekyll serve

  🚀 部署选项

  GitHub Pages（推荐）：

  - 推送到 main 分支后自动部署
  - 免费托管，支持自定义域名
  - 内置 SSL/HTTPS 支持

  自定义域名：

  1. 添加 CNAME 文件，填入你的域名
  2. 配置 DNS 记录
  3. 在 GitHub 设置中启用 HTTPS
